# 🧑‍💻 Jegadeesh (Jack) Nandakumar — Portfolio

> **Live Site →** [jxck007.github.io/portfolio](https://jxck007.github.io/portfolio)

A dark-themed personal portfolio built with vanilla HTML, CSS, and JavaScript. Features live GitHub and LeetCode stats, animated UI, and a fully responsive layout — no frameworks, no build tools, single file.

---

## ✨ Features

- **Live GitHub Stats** — repos, followers, language breakdown, contribution graph & streak
- **Live LeetCode Stats** — easy / medium / hard counts with animated SVG rings, tries 3 APIs with fallback
- **All-in-one** — single `index.html`, zero dependencies, no build step
- **Smooth animations** — scroll reveal, floating glows, animated language bars
- **Responsive** — works on mobile, tablet, and desktop

## 🚀 Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | HTML5, CSS3 (custom properties, grid, flexbox) |
| Fonts | [Syne](https://fonts.google.com/specimen/Syne) + [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) |
| Stats | GitHub REST API, LeetCode API (3-endpoint fallback) |
| Widgets | [ghchart.rshah.org](https://ghchart.rshah.org), [leetcard.jacoblin.cool](https://leetcard.jacoblin.cool), GitHub Readme Streak Stats |
| Hosting | GitHub Pages |
